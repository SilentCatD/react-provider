# React Scoped Provider

I have always like the idea of scoped dependency injection, simply because of it provides comprehensible resources release mechanism.

A resources assiosiated with a life cycle of a view should also be disposed when the view is unmounted.

## Feature

- Dependency Injection without the need to create `Context` manually for each `DataType`
- Handled `Context` / `Provider` hell scenario
